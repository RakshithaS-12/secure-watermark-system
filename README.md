# 🔐 Secure Authentication System

A mini project that uses **Digital Watermarking (LSB)** and **SHA-256 Hashing** to ensure document authenticity and detect tampering.

---

## 📌 Overview

This system protects digital images by:

- Embedding a hidden watermark inside the image
- Generating a SHA-256 hash for integrity verification

If the document is modified, the system detects it by comparing hash values.

---

## 🎯 Features

- Upload image/document
- Embed hidden watermark (LSB technique)
- Generate SHA-256 hash
- Store hash in database (SQLite)
- Verify document authenticity
- Detect tampering

## ▶️ How to Run

1. Clone the repository:
git clone https://github.com/RakshithaS-12/secure-watermark-system
cd secure-watermark-system
2. Install dependencies:
pip install -r requirements.txt
3. Run the application:
python app.py
4. Open in browser:
http://127.0.0.1:5000

## 🔄 Working

### Upload Phase
- Upload image
- Generate SHA-256 hash
- Embed watermark
- Store hash in database

### Verification Phase
- Upload image again
- Generate new hash
- Compare with stored hash
- Detect tampering
